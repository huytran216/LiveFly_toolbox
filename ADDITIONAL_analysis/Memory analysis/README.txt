This folder contains the script to probe for any kind of relationship between the mothers' and daughters' features.
1. Run main_extract.m
    Select the dataset and run the analysis.
2. Run main_show_feature.m
    Show the features of the nuclei in different cell cycles.
    Reevaluate how accurate the gene/nuclei senses its position along the AP axis.
3. Run probe for memory
    Remember to select the time series and cycle to test first
    Look at imgrec folder for results, open images with imageJ
